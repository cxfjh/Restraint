# 自律锁机 - 专注时间管理工具

## 应用介绍

自律锁机是一款帮助用户专注工作、学习，避免手机干扰的应用。通过设置锁定时间，应用会限制手机的使用，帮助用户保持专注，提高效率。

## 功能特点

- 设置自定义锁定时间，在设定时间内限制手机使用
- 采用无障碍服务防止用户绕过锁定
- 网络拦截功能减少消息干扰
- 支持开机自动恢复锁定状态
- 简洁直观的操作界面

## 开发环境

- 编译SDK版本：36
- 最低支持SDK版本：29
- 目标SDK版本：36
- 主要依赖：Material Design、AppCompat
